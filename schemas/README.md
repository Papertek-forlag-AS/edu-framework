# JSON Schemas

This directory contains the contract layer for the Papertek Framework for Education.
AI agents and validation scripts use these schemas to ensure content conformance.

## Schema Files (Phase 1 deliverables)

- `exercise.schema.json` — All 22 exercise types (most complex, build first)
- `vocabulary.schema.json` — Core vocabulary entries + translations
- `lesson.schema.json` — Lesson structure (dialog, goals, grammar topics)
- `grammar.schema.json` — Grammar module definitions
- `curriculum.schema.json` — Curriculum metadata and configuration
- `question-bank.schema.json` — Test/quiz question definitions
- `language.schema.json` — Language-specific configuration
