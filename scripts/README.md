# Build Scripts

Validation and build scripts for the Papertek Framework for Education.

## Scripts (Phase 1 deliverables)

- `validate-schemas.js` — Validate content against JSON schemas
- `validate-exercises.js` — Check exercise ID uniqueness and counts
- `validate-vocabulary.js` — Check for orphan words
- `validate-lessons.js` — Verify generated lessons match templates
- `build-vocabulary.js` — Compile vocabulary banks
- `generate-lessons.js` — Generate lesson HTML from templates + data
- `dev-server.js` — Local development server
